# Store-Sales-Analysis-PowerBI

🔍 Project Overview       
This project presents an interactive Power BI dashboard analyzing store sales data to uncover trends in revenue, regional performance, and overall business growth.      

🎯 Objective        
To transform raw sales data into meaningful insights that support data-driven business decision-making.

📊 Key Insights              
Identified top-performing regions contributing to revenue               
Analyzed monthly and yearly sales                     
Highlighted high-performing product categories                  
Tracked overall sales performance using KPIs        

🛠 Tools & Technologies         
Power BI     
DAX (Data Analysis Expressions)                
Data Modeling          
Data Cleaning             
Interactive Dashboard Design     

💡 Business Impact          
Helps in tracking sales performance         
Supports strategic decision-making         
Identifies growth opportunities
